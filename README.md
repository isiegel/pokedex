# Pokédex

A Next.js app that displays random Pokémon and their details, powered by [PokéAPI](https://pokeapi.co) via [pokenode-ts](https://github.com/Gabb-c/pokenode-ts).

## Features

- Home page with 20 random Pokémon — refresh for a new set
- Detail page with stats, abilities, height, weight, and official artwork
- Server components with streaming and loading skeletons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Testing

```bash
# Unit tests (Vitest)
npm test

# E2E tests (Playwright) — starts the dev server automatically
npm run test:e2e
```
